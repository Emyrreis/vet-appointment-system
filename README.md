Um sistema simples de agendamento de consultas veterinárias desenvolvido em python, aplicando conceitos de Programação Orientada a Objetos e Padrões de Projeto para o meu
trabalho final na disciplina. 

Funcionalidades:

- Cadastro de clientes e pets
- Cadastro de veterinários
- Agendamento de diferentes tipos de consultas (rotina, emergência, vacinação)
- Sistema de notificações por Email e SMS
- Gerenciamento de status das consultas

Tecnologias Utilizadas:

- Python
- Typing (para type hints e protocols)
